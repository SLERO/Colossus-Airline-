Program for booking tickets for 4 flights. On exit, saves the data to a file. At startup, unloads info from a file. Allows you to select a flight and then:
1) View the number of free seats
2) View the list of free seats and reserved seats
3) View the list of reserved seats sorted in alphabetical order
4) Book a place. Overwrite protected.
5) Remove the armor from the place
6) Clear file for the selected flight + confirmation request to erase data.
